Add a new query parser¶

Involved Files¶

QueryParser.class.php, UssContainer.php

Mechanism¶

  1. Create a new class in /USS/parser/
  2. The new class should exntends QueryParser, for example ‘newParser.class.php’
  3. Open ussContainer.class.php, and add the dependency by adding “include_once parser/newParser.class.php;” statement in it
  4. Add a case block of the new class into getParser() method’s switch block

Template¶

Add a new tokenizer
